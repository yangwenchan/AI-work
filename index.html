🚀 AI Prompt 智能寶庫 · 以 GitHub Pages 部署

關於這個專案 🧩

AI Prompt 智能寶庫是一個開放、持續更新的提示詞資源平台，讓每個人都能善用 AI 工具。

🚀 這是什麼？

  • ✍️ 內容創作 — 文案、品牌命名
  • 📝 筆記精煉 — 書籍摘要整理
  • 💻 程式應用 — 程式生成、除錯
  • 📊 簡報表達 — 結構化提煉
  • 🎨 圖片生成 — AI 繪圖提示詞
  • 🗂️ 專案企劃 — 計畫書規劃

☁️ 資料來源

所有 Prompt 儲存於 Google Sheets，網站載入時直接從雲端讀取，無需伺服器。

試算表需設定為「知道連結的人可以檢視」才能正常讀取。

📖 如何使用

1
前往 Prompt 庫點上方導覽「Prompt 庫」
2
搜尋或篩選輸入關鍵字，或點分類標籤
3
複製 Prompt按「複製 Prompt」按鈕
4
貼到 AI 工具貼到 ChatGPT / Claude 即可

⚙️ 技術架構

  • 🏗️ 純 HTML / CSS / JS
  • ☁️ Google Sheets CSV API
  • 🌐 GitHub Pages 靜態託管
  • 🔤 Google Fonts 字型

更新試算表後重新整理即可看到最新內容。
